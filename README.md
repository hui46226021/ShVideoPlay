# ShVideoPlay
android 视频播放封装
Download the latest JAR or grab via Maven:
~~~xml
<dependency>
  <groupId>com.sh.zsh.code</groupId>
  <artifactId>shvideoplayer</artifactId>
  <version>1.0.2</version>
  <type>pom</type>
</dependency>
~~~
or Gradle:
~~~gradle
compile 'com.sh.zsh.code:shvideoplayer:1.0.2'
~~~
